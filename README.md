## About This Project

This is my first web development project — a simple e-commerce platform inspired by thrift stores.

Users can:
- Donate items they no longer need
- Browse and purchase items donated by other users

The application is built using Flask, MySQL, HTML, CSS, and JavaScript.

During the development of this project, I relied heavily on AI-assisted tools to help generate and structure parts of the codebase. This allowed me to focus on understanding the architecture, database interactions, and overall workflow of building a full-stack application.

This project represents an early step in my learning journey, where I explored how modern AI tools can assist developers while building and experimenting with real applications.
